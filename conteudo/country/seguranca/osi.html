#Segurança da Informação[OSI]

$Open Systems Interconnection

Para entender melhor como as conexões na internet funcionam, é necessário compreender as camadas do modelo de referência para interconexões de sistemas abertos (OSI). É um modelo conceitual que descreve as funções de um sistema de comunicação ou rede em sete camadas distintas. O objetivo do modelo é guiar a implementação dos padrões de rede para permitir cooperação entre diferentes produtos de softwares de diversos fabricantes.

$Camadas do OSI

Física/
Trata da transmissão de bits brutos através de um físico. Envolve hardware como cabos, switches e placas de rede.
Ligação%de%dados/
Estabelece a ligação entre dois 'nós' fisicamente conectados. Essa camada é responsável pela correção de erros que podem aparecer na camada física, e usa protocolos de ethernet, HDLC e MAC.
Rede/
Gerencia o roteamento de dados entre redes diferentes, assim como os endereços para cada rede e sub-rede. Essa camada usa os protocolos IP, já explorado, e o ICMP.
Transporte/
É a camada que garante o envio e recebimento dos pacotes vindos da camada três. Ela lida com a qualidade de serviço para que os dados sejam entregues com consistência, sem erros ou duplicações. Os protocolos comuns dessa camada são o TCP, também já explorado, e o UDP.
Sessão/
Essa camada é responsável por estabelecer e encerrar a conexão entre os host's, ou seja, ela faz a sincronização dos mesmos. Protocolos como SMB e RPC.
Apresentação/
Essa camada é responsável pela tradução dos dados para a próxima camada. Ela converte os códigos para caracteres, como também compacta os dados e os criptografa.
Aplicação/
A camada de aplicação é onde temos os dados de consumo para o usuário através dos softwares como os navegadores. Essa camada usa protocolos como o HTTP, STMP e DNS.

//Progressão

Por enquanto vamos parar um pouco com essa parte mais voltada a redes de comunicação e começar a explorar outras áreas mais voltadas ao usuário final. Quando formos tratar das ferramentas retornaremos ao assunto de protocolos, pois serão importantes e ainda não foram abordados em sua totalidade.
