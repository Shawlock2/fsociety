#Segurança da Informação[TCP-IP]

$Introdução

Transmission Control Ptotocol/Internet Protocol ou apenas TCP/IP se trata do conjunto de protocolos e regras que permitem a interligação dos dispositivos pela internet. Em poucas palavras, é o protocolo que realmente faz a internet funcionar. Ele é responsável pelo envio, roteamento e recepção dos dados e funciona através de diversas camadas.

$Camadas

Aplicação/
É a camada responsável pela interação direta entre os aplicativos e serviços de rede. Essa camada usa os protocolos HTTP, FTP e SMTP
Transporte/
Essa camada tem por função garantir a transferência dos dados entre os Host's e estabelecer corretamente as conexões e o controle do fluxo de dados. Essa camada pode usar tanto o próprio TCP, para garantir a entrega ordenada e sem erro dos dados, quanto o UDP, que embora seja mais rápido que o TCP, não é tão confiável quanto a garantia de entrega de dados.
Internet/
Nessa camada os endereços IP são definidos e os pacotes de dados são encaminhados entre diferentes redes. O Internet Protocol é responsável pelo endereçamento e roteamento dos pacotes de dados, podendo ser a versão IPv4 ou a versão IPv6. E o Internet Control Message Protocol é usado nessa camada para exibir diagnósticos, como o comando ping.
Rede/
É a camada que trata do acesso físico de acesso aos dados, a exemplo de redes de ethernet e redes Wi-Fi.
